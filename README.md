# AssetsServer

  Compiles and serves static assets
  Renders templates on demand


## Installation

Add this line to your application's Gemfile:

    gem 'assets_server'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install assets_server

## Usage

    require 'assets-server'
    run AssetsServer.new(
      # options
    )
